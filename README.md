# hthvote-guide

## 项目简介

本项目是一个用于归档和发布独立 HTML 页面的仓库。其中包含的页面均为独立的静态 HTML 文件，不依赖于特定域名或网站环境，可直接在浏览器中打开查看。

这些页面主要用于整理和展示某些流程指引、操作说明或信息参考，内容结构清晰，适合作为离线或在线参考文档使用。

## 目录说明

```
hthvote-guide/
├── pages/           # 存放所有独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/          # 可选：页面引用的静态资源（如图片、样式表等）
│   └── ...
├── README.md        # 本文件
└── LICENSE          # 开源许可文件
```

- `pages/` 目录下存放所有已归档的 HTML 页面，每个页面均为完整自包含的文档。
- `assets/` 目录可存放页面所需的外部资源文件，如样式、脚本或图片等（如适用）。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务或数据库。
- 页面内容在归档时已确保格式正确，可直接通过浏览器打开使用。
- 部分页面可能引用本地资源，建议保持目录结构完整以确保正常显示。
- 本仓库不针对任何特定域名或网站，页面内容仅作为通用指引或信息展示。

## 维护说明

- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保其内部链接及资源引用路径正确。
- 建议在提交前检查页面在不同浏览器中的显示效果。
- 若页面引用外部资源（如 CDN 库），请尽量选择稳定版本，并注明来源。
- 本仓库欢迎提交改进或修正，请遵循常规 Pull Request 流程。
